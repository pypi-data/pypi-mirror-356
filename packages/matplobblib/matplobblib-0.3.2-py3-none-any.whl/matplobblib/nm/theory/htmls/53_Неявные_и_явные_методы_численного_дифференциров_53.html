53. Неявные и явные методы численного дифференцирования.

Начиная со слайда 375

  •  Явный метод
  • Метод называется явным, если значение на следующем шаге вычисляется только через известные значения с предыдущих шагов.
  • Формула: y_{n+1} = y_n + h*f(t_n, y_n)
  • Пример: Явный метод Эйлера

Для уравнения dy/dt = f(t, y):         y_{n+1} = y_n + h * f(t_n, y_n}

  • Неявный метод
  • Метод называется неявным, если значение y_{n+1} зависит от самого себя, т.е. оно входит в правую часть формулы.
  • Формула: y_{n+1} = y_n + h*f(t_{n+1}, y_{n+1})
  • Пример: Неявный метод Эйлера

Для уравнения dy/dt = f(t, y):         y_{n+1} = y_n + h * f(t_{n+1}, y_{n+1})
