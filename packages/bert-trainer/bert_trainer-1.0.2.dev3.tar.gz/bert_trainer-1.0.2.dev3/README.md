## Run Locally

Clone the project

```bash
  git clone https://gitlab.cyberspike.top/aszl/diamond-shovel/al-1s/bert-trainer
```

Go to the project directory

```bash
  cd bert-trainer
```

Install dependencies

```bash
  pip install .
```

Start the server

```bash
  bert-trainer --help
```
