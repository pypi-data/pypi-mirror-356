Accessibility

Select accessibility options

Select accessibility settings below as require to find a setup you like. Once the appearance of the page is satisfactory, click on the "Save" button to apply the settings to the entire website. This saves the settings in a cookie on your device which is only used for this purpose and does not store any personal data.

The colour options here are not compatible with the dark-mode toggle in the header bar. If a custom dark scheme is desired, the appropriate colours can be selected in light-mode.
